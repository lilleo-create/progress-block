# Progress Block
Прототип блока Progress для мобильных web-приложений

## Демо 
[Открыть демо](https://lilleo-create.github.io/progress-block/)

## Функциональность
- Normal — управление размером дуги через Value (0-100)
- Animated — вращение блока по часовой стрелке
- Hidden — скрытие блока со страницы
- Адаптация под ориентацию экрана (portrait/landscape)

## API
```js
Progress.setValue(75)      // установить прогресс 0-100
Progress.setAnimated(true) // включить/выключить анимацию
Progress.setHidden(true)   // скрыть/показать блок
```

## Технологии

- JS (Vanilla JS, без библиотек)
